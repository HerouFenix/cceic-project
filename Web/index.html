It's all about structure!

In this website you will find carefully chosen content, organised by chapter, so that you can choose your own path and quickly access information in a visual and atractive way.

Curated Content

All the available content was carefully chosen and it's referenced correctly. By using this approach, the reader can easily be provided with information that is already proven to be correct and quickly investigate the sources behind such content.

Digestible Chapters

The content is organised by the use of chapters. With this structure type, it is easier to plan a path to be followed in order to better define a learning experience. It is also easier to find a specific topic.

Exercises

Since learning without practice is a rather difficult achievement to be taken, a set of exercises can be found at the end of each chapter so that the knowledge previously gathered can be applied and solidified.
